### Gallery page

On Gallery page you can create new Galleries and Albums, upload(add) and remove images, edit images names and descriptions.
This is the main page from where you can control your Gallery and Album elements. Below you can see how to create Gallery, Album and how to upload images. 

First click on "Create gallery", then type the Gallery name and click on "Add new gallery" button. After that you will see the message "Gallery successfully created".
Now you can create albums for new created Gallery.
Click the Plus button on the left side of created Gallery and you will see "Create album" option. Click on "Create album" and type there the album name and then click on the Create album button on the right side.
After this steps you will see "Album successfully created" message.
Then click the Plus button on the left side of created Album and you will see "Add new image" option. Click on Add new image and you will see the upload option in the right side.
You can use the WpDevArt uploader for uploading new images or choose images from WordPress media library. Just choose or upload images and click on Insert to album button from bottom.
If you want to edit the Album images, then click on the square in left side of Album name, if you want to edit Album name or delete the Album, then click on the Album name.

### Adding Gallery into your website posts/pages

After you create Gallery, configure Popup and other settings, you need to go to your posts/pages for adding Gallery.
Find the insert Gallery shortcode button from Visual Editor buttons(search for Gallery icon). Click on it, then in opened popup select the Gallery(or you can select all),  then select the Album from selected Gallery(again you can select all), after that select the Gallery theme you want. Then choose the Gallery view you prefere and click on Insert button.
If you want to edit Gallery view or other options, then click on icon that was added to the post/page after inserting Gallery and you can change the options.
